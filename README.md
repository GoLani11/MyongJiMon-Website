# Myong Ji Mon - Website

25년 명지전문대 캡스톤 디자인 "명지몬" 웹사이트 졸업 작품 제작 프로젝트입니다.
React, Node.js, MySQL을 사용하며, Azure 서버에 배포합니다.

## 프로젝트 개요
- **목표**: 대학 웹사이트를 통해 학생 및 교직원에게 편리한 서비스 제공
- **기술 스택**:
  - Frontend: React
  - Backend: Node.js
  - Database: MySQL
  - Server: Azure

## 프론트
cd frontend
npm install
npm start

## 백엔드
cd ../backend
npm install
node index.js
